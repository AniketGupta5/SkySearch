# Geographical Expansion Plan — SkySearch

## Current Reach
- Supports all international and domestic flight routes globally
- Available in 3 languages: English, Hindi, and Telugu
- Accessible via web browser — no installation required
- Live at: https://skysearch-mq5stdnqm2hady7uhpyuzh.streamlit.app/

---

## Phase 1: Indian Market (Current)
- **Target users:** Indian travelers, students, budget travelers
- **Languages supported:** English, Hindi, Telugu
- **Popular routes pre-loaded:** Delhi, Mumbai, Bangalore, Hyderabad, Chennai, Kolkata, Goa
- **Onboarding:** Instant via shareable link — zero friction

## Phase 2: South & Southeast Asia Expansion
- Add Tamil, Kannada, Malayalam language support
- Add popular Southeast Asian routes (Singapore, Bangkok, Kuala Lumpur)
- Localize currency display for SGD, MYR, THB
- Target users in Sri Lanka, Bangladesh, Nepal

## Phase 3: Middle East & Global NRI Diaspora
- Add Arabic language support
- Focus on UAE, Saudi Arabia, Qatar routes (heavy Indian diaspora)
- Add AED currency (already supported)
- Partner with NRI student communities abroad

## Phase 4: Global Rollout
- Add support for 10+ languages via LLM multilingual capabilities
- Expand to European and North American routes
- Add local airline suggestions per region

## Horizontal Scaling Strategy
- App is fully cloud-hosted on Streamlit Cloud — scales automatically
- No backend infrastructure to manage
- New language support requires only prompt engineering changes
- New routes added via IATA code database updates

## Onboarding Strategy
- Single shareable URL — works on any device, any browser
- No login or signup required
- In-app language switcher for immediate localization

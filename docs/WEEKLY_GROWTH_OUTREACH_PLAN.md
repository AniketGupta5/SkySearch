# Weekly Growth & Outreach Plan — SkySearch

## Objective
Grow SkySearch's user base, improve features continuously, and increase engagement through structured weekly outreach.

---

## Week 1–2: Internal Launch & Feedback Collection
- Share app link within BITS Goa student community and Swecha PS1 cohort
- Collect initial feedback via the in-app Google Sheets review system
- Fix critical bugs based on early user feedback
- Goal: 10+ reviews collected

## Week 3–4: Feature Improvement Sprint
- Analyze feedback from Google Sheets
- Improve flight result accuracy based on user suggestions
- Add more airline booking links
- Improve UI responsiveness on mobile
- Goal: Deploy 2–3 feature improvements

## Week 5–6: Social Media & GitHub Outreach
- Post on LinkedIn with demo video and app link
- Share on GitHub with updated README and screenshots
- Post in relevant Reddit communities (r/india, r/travel, r/learnpython)
- Goal: 50+ app visits

## Week 7–8: Cross-Platform Expansion
- Reach out to travel communities and student groups across other PS1 stations in Hyderabad
- Share with BITS alumni network
- Submit to Streamlit app gallery
- Goal: 100+ app visits, 20+ reviews

## Continuous Improvement Strategy
- Monitor Google Sheets reviews weekly
- Prioritize feedback by frequency and impact
- Use GitLab CI/CD pipeline (5 stages) to ensure quality before every deployment
- Track issues via GitLab issue tracker

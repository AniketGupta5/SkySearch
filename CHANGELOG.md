# Changelog

All notable changes to SkySearch will be documented in this file.

## [1.0.0] - 2026-06-11

### Features
- AI-powered flight search using Groq (Llama 3.3 70B)
- Search any route by IATA codes
- Multi-currency support (USD, INR, EUR, GBP, AED, SGD)
- Economy, Business, First class filtering
- Sort by price, duration, or departure time
- Stats bar showing cheapest, average, most expensive fares
- AI travel recommendation for each search
- Dark sky-themed UI
- Quick route picker for popular Indian and international routes

### Bug Fixes
- Fixed HTML rendering in flight cards
- Updated Groq model from deprecated llama3-70b-8192 to llama-3.3-70b-versatile

# Contributing to SkySearch

Thank you for your interest in contributing!

## Getting Started
1. Fork the repository
2. Clone your fork: `git clone https://code.swecha.org/YOUR_USERNAME/aniketgupta_skysearch`
3. Install dependencies: `pip install -r requirements.txt`
4. Run locally: `streamlit run app.py`

## Making Changes
- Create a new branch: `git checkout -b feat/your-feature`
- Make your changes
- Run linting: `ruff check app.py`
- Commit using conventional commits: `git commit -m "feat: add new feature"`
- Push and open a Merge Request

## Code Style
- Python 3.11+
- Ruff for linting
- Mypy for type checking
- Follow existing code structure

## Reporting Issues
Open an issue on the GitLab repository with a clear description.
